1.Программа работает на языке Python 3.x

2.Нужны следующие библиотеки:
	requests, BeautifulSoup, lxml, pymysql.
Все устанавливаются через "pip install <название_библиотеки>" в командной строке.

3.Нужна подготовленная база данных MySQL. 
	Table_name avito
	Columns: name,price,about,user_info,user_link,user_id,about2,picture
Колонки с кодировкой utf8_general_ci. С длиной сами разберетесь, некоторые объявления массивные.

4.Конфиг БД прописать в файле parser.py в 81 строке кода.

5.Запуск либо через командную строку python avito.py
либо через IDE. 

Все работает как часики, парсит и 30 и 50 и 100 страниц, в зависимости какую конечную страницу задать в 107 строке кода.
На авито максимальная страница по спецтехнике под номером 100. 

Если не парсится id пользователя с линком на него, то это рекламное объявление скорее всего.
